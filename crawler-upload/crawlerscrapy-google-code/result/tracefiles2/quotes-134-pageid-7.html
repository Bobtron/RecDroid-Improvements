ctouch

smartphone simulator (or emulator) extention for webkit browsers (Chrome / Chromium / SRWareIron / ComodoDragon / Maxthon / Safari / Opera)

Migrated to https://github.com/cielavenir/ctouch/

  • 2014 Oct 23:

    • Sound test: http://areweplayingyet.org/ https://github.com/soundcloud/areweplayingyet/
    • For example, gree::58737 uses aac(m4a) and gbf.mbga uses mp3.

  • 2014 Sep 8: Migration done successfully.

    • As I said before, google code downloads stopped working. Further off-store release will be pushed to sourceforge.
  • 2014 Aug 31: We are migrating to https://sourceforge.net/projects/ctouch/. I hope I can release next version (as binary) soon.
  • Supported browsers: Chrome / Chromium / SRWareIron / ComodoDragon / Maxthon / Safari / Opera 15+)
    • Maxthon: Use classic version. Then download crx (hosted on sf.net) to install cTouch.
    • Opera 15+: https://code.google.com/p/ctouch/wiki/OperaInstall
    • Safari: https://code.google.com/p/ctouch/wiki/SafariInstall
    • Others: https://code.google.com/p/ctouch/wiki/ChromeInstall
      • According to cTouch Test Suite, gree::2826 is unplayable on Safari and Opera.
  • This extension emulates some JavaScript properties to mimic iPhone/Android.
  • They will think you are using smartphone via Wi-Fi...

    • 本変更はcTouchクローンを抑制するためのものです。研究目的での利用の場合は「cTouch由来の部分が【主】となる」ことはまず起こらないでしょう。

    • パスワードフィールドをテキストフィールドに切り替えるブラウザアクションを追加します。

  • LinkSource (Add context menu (link) to Open link with view-source)
    • リンクコンテキストメニューに「Open link with view-source」を追加します。
