matted-blues-sense

htc evo based sense theme

This is a project page for the MattedBlues Sense version. MattedBlues was a theme created by Beatblaster and mackgk1. I did NOT create the theme from scratch, full credit goes to them for this awesome theme. I am simply porting this theme over to Sense.

This theme is being tested on TommyTomatoes EVO Classic ROM. You can download this rom at the link below:

http://forum.xda-developers.com/showthread.php?t=1312414

Please report ANY issues or missing PNG's here so I can keep track of any missing images, as well as update the changelog for the version.

Themed icons will be rolled out with the 1.4 update for those wondering.
