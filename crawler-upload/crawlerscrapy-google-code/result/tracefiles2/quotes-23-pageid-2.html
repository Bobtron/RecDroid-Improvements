wireless-mapping-and-localization-with-smartphones

Wireless mapping and localization using smartphones

Using available wireless signals inside of a space in order to localize a mobile robot or anything else that has wireless signal receptors.

Initially using a smartphone, a grid will be created to capture the room's dimensions. This grid will be split in cells, and afterwards each cell will contain the wireless information captured by the device (signal strength and ssid). Based on the readings afterwards, the phone will be able to say it's current location inside of the grid.

Later on an alternative method will be used for mapping the room or the entirety of the space.
