ucxl on HTC Desire HD/Inspire 4g

HTC Desire HD Sense4.1 SP10X Collector's Edition (收藏版)

系统介绍：

Android Version: 4.0.4

Based on: One V 2.22.405.1 & One st & Desire x

Sense Version: 4.1

Kernel Based on: Flemmard kernel added kangbang kernel New features

Supported Devices: Desire HD / Inspire 4G

更新日志：

1、本不想出完整包，但为考虑机友升级方便，本版本加入OTA更新（只要有网就可以检测有无更新），像HTC官方一样支持自动下载更新，方便快捷，为本系统以后的持续升级提供更大的保障。此版为收藏版，以后我会以OTA推送的形式给大家带来更新。。。

2、内存控制更加完美

3、集成新的S7内核

4、修复系统音效部分失效问题（原音效最下面几个音效无效）

5、新的毒蛇商店

6、各种动画完美(感谢孤灵泪机友提供)（不喜欢的可以在“设置”-“显示”去掉动画） 7、其他软件升级最新版

注意：

1、一直有人问自带音乐怎么显示歌词，这个要有前提，第一要是mp3格式的。音乐文件内含歌曲信息（如艺术家，歌曲名），然后更新专辑封面就能显示歌词了。

2、如果无法使用ota全自动更新的朋友可以先更新recovery到最新版本（触摸版），然后再用ota更新。

3、实在无法ota的也可以选择下载对应的更新包手动更新:http://code.google.com/p/ucxl/downloads/list

4、天气源已经ota推送为accu的了（天气准确度高，但不能用移动网络更新），需要中国的天气源的可以下载国内天气源直接刷入。国内天气源下载地址（可用移动网更新，但天气不准确）：http://ucxl.googlecode.com/files/ChinaWeather.zip

Sense4.1 SP10X完整包下载地址：http://pan.baidu.com/share/link?shareid=200214&uk=3858779036
