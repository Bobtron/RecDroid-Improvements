Hatcam

Hatcam ist ein Android-Softwareentwicklungsprojekt an der Fachhochschule Technikum-Wien mit dem Ziel der Implementierung auf einem IGEPv2 Embedded Board.

Projektbeschreibung

Hatcam ist eine Android Anwendung welche auf einem IGEPv2 Embedded Development Board aufgesetzt werden. Als Betriebssystem dient ein für unser Board modifiziertes Rowboat Froyo 2.2 welches um WLAN-Funktionalität sowie die Anbindung an eine USB-Kamera erweitert wurde.

Hatcam ist eine Anwendung welche in regelmäßigen Zeitabständen Fotos mit der Kamera schießt, und diese auf einem externen Speichermedium hinterlegt. Sobald ein Internet-Zugangspunkt (in Form von Wifi-Access-Points) in der Nähe ist, beginnt Hatcam automatisch mit dem Upload der Fotos auf eine Web-Plattform.

Zielplattform

Die Zielplattform ist das Embedded Development Board IGEPv2 welches uns in der Revision B vorliegt.

  • IGEPv2 Revision B
  • Rowboat Froyo 2.2 (Android API-Level 8)
