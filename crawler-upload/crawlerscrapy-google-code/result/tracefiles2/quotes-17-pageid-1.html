j2memaprouteprovider

small class to use in j2me, Android, BlackBerry, get a route by start and end locations

uses http://maps.google.com'>http://maps.google.com'>http://maps.google.com to route a driving directions between two locations and parse response http://code.google.com/intl/uk/apis/kml/documentation/'>kml file with SAX parser into easy coordinates structure. Contains example applications on Blackberry and Android.
