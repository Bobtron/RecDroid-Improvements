droid-utils

Android Utilities

WARNING: These utilities are currently in beta. Please http://code.google.com/p/droid-utils/issues/list'> file bugs!

Utilities that help in the development of Android application. This library currently consists of one individual tools for developers:

1. Android JSON Utils

Try out the current version: http://code.google.com/p/droid-utils/downloads/detail?name=android-json-utils-0.1.jar&can=2&q='> Android Json Utils

Android JSON Utils is a android library that can be used to convert Java Objects into their JSON representation. It can also be used to convert a JSON string to an equivalent Java object. Android JSON Utils can work with arbitrary Java objects including pre-existing objects that you do not have source-code of.

There are a few open-source projects that can convert Java objects to JSON. However, most of them having their own json parser and its not optimized to android vm and its added extra coded to application. Android JSON Utils considers both of these as very important design goals.

Features

  • Lightweight component.
  • No external parser its required because its use default android json parser.
  • Provide simple deserialize () and serialize() methods to convert Java objects to JSON and vice-versa
  • Allow pre-existing unmodifiable objects to be converted to and from JSON
  • Extensive support of Java Annotation.
  • Customized date format , we can convert the json date format into user defined date format at parsing time.
  • Customized json array size, its will help you to avoided unwanted data’s in the collection data type , i.e suppose you have 100 elements in the json array response , but you need only 10 elements on the UI, by using @ItemType(size=10) annotation you can archive this.
  • Mapping json key name into customized java object field name , meaning dont need to keep same json key name as a field name , if you want different name from json key then you can use @SerializedName(“name”)
