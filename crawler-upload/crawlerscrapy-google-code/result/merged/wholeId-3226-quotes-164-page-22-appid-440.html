Comment #1

Posted on Feb 14, 2012 by Quick Hippo

Is there another sip application installed on your phone?

There is many sip providers that does a rebranded version of csipsimple. Basically they just reuse my code and add their logo and a custom wizard. That's authorized by my GPL license provided the fact they announce the fact their application is also GPL and based on CSipSimple source code. The list of known redistribution that respect the license model : http://code.google.com/p/csipsimple/wiki/BrandedVersionsOfCSipSimple?wl=en

Please tell me if you have another sip application that does not respect my license. I'll ask them to make things in respect to the license. The aim of my license is that it will benefit all users (and that users are free to redistribute the app, modify and study the source code). If one redistribution break that idea, I can ask for the removal of such an app.

Else, it's just normal, and it's good for the project that other companies rely on the project.

Anyway, CSipSimple must overlap all features of any sip provider rebranding (because all rebrand must contribute new features ;) ). So you can use CSipSimple as soon as you have more than one sip provider. Besides, CSipSimple (particularily nightly builds) will always be more up to date than any rebranded version ;)
