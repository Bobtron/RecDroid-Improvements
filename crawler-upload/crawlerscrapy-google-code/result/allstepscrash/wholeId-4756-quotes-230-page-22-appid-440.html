Comment #1

Posted on Jul 1, 2010 by Quick Hippo

Hi,

Did you tried with the latest 0.00-12 pre build or with the version available on the android market? Besides, two things : * sip.goandcall.com port 6060 is not open (at least, as far as I can see from my pc) * In your case, I think that Proxy URI field is useless : the only document I found on the web about xelow say to configure a proxy but in fact it is simply a part of the Registration uri used for the registration. So you can probably leave blank the sip proxy field. You can probably also use the Basic account wizard that is more easy to configure : You'll have to fill your username, your password and "sip.goandcall.com" as domain name.

Let me know if any of theses advise solve the problem for you.
