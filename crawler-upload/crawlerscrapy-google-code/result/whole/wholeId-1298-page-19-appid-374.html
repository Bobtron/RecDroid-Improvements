Comment #8

Posted on May 25, 2012 by Grumpy Rhino

The Network Log app uses the Linux iptables infrastructure to log network connections. Unfortunately, this requires root/superuser access.

Going forward, the app will now check for root/superuser access before attempting to invoke iptables -- and display an informative dialog if the device is not rooted.
