csipsimple - issue #1505

Enhancement when Answering Incoming Video Calls

Posted on Jan 16, 2012 by Helpful Cat

What steps will reproduce the problem?

  1. Install latest nightly build
  2. Receive a video call

What is the expected output? What do you see instead?

If you have video enabled and you receive a video call the user should still have the ability to reject video and answer the call as an audio-only call.

It would be nice if InCallActivity2 had two (2) answer buttons only when video is enabled in the call settings. The first answer button would accept the call with audio only and the second answer button would accept the call with audio+video.

What version of the product are you using? On what operating system?

r1208 / Samsung Galaxy Tab (Android 2.2) Please provide any additional information below.

Comment #1

Posted on Jan 16, 2012 by Quick Hippo

Yes it's a solution. If it's the chosen way, it also means that we should add a video call button for parity everywhere.

Another idea I had was to allow user to activate video after he answered. (just like the speaker feature actually). And to have a global option to tell that we always want to use video.

I was more envisaging the second one, because I want to try to keep the UI as basic as possible. And when you want to answer a call you don't really yet mind about video or audio. Then, later the user can decide to switch to video anyway. Or if he want to use the application in video mode, he can enable call/answer with video as a global setting.

I also have in mind that I've another application in my boxes that I'll release soon to automatize lot of things on android and as a side plugin things in android. It would enable to automatically change settings (such as this one for video). So that user does only have to choose once for example if he want to reply with video when he is on such or such wifi ssid. And then don't have to bother for each call. (or only if he decides to do something manually).

But indeed, there is something to do anyway :)
