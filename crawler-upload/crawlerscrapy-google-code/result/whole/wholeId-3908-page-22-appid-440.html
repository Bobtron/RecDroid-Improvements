Comment #9

Posted on Nov 6, 2011 by Happy Monkey

Ok, works now, you probably meant to set the "Clock rate" to 32 KHz. I'm rather sure I tried that before but maybe for unknown reasons a reboot was necessary to make it really work.

For other users with GT-P1010 the following settings work for me (others may or not): Clock rate: 32 KHz Use WebRTC implementation: off Use routing API: off Use Mode audio API: off Use Samsung Galaxy Hack: off(!) Tone hack: off Focus audio: on Micro source: default
