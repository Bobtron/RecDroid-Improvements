[wizard] add Hibritel

Posted on Sep 6, 2013 by Swift Panda

Hello and thanks for your great work on csipsimple!

The service is based in Spain.

The customer has to enter the following information:

Username: alphanumeric string Password: alphanumeric string

Predefined values: - All the options by default in account settings, except:

 Account name: hIBRITEL  Registration URL: sip:10.200.1.254  Realm: *  Transport: UDP  Register time out (sec): 600  Proxy Url: sip:10.200.1.2524  Port RTP: 7100  QoS: Force  DSCP: 46  Allow SDP NAT Rewrite: enabled

  • All the options by default in settings, except:  Multimedia  Codecs: Only Speex selected in fast and slow tabs  Speex 32 kHz  Speex 16 kHz  Speex 8 kHz  Easy configuration: Only selected Available on WIFI (Calls only work via Wifi)

The logos can be found here:

Logo PNG 96x96 "HIBRITEL": http://www.randomiberica.com/hibritel96_96.png

Please let me know if you need more information for preparing the wizard.

Thanks in advance!
