android-on-freerunner - issue #12

Should properly display unsolicited USSD messages

Posted on Sep 22, 2009 by Happy Rabbit

What steps will reproduce the problem? 1. USSD (http://en.wikipedia.org/wiki/Unstructured_Supplementary_Service_Data) is the way GSM phones query and report status information, like prepaid balances. 2. Currently USSD messages are displayed as a string of hex digits in a dialog box.

What is the expected output? What do you see instead? They should be properly decoded into readable text.
