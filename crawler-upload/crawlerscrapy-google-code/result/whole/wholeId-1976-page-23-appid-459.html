Comment #2

Posted on Aug 19, 2011 by Quick Giraffe

What you're seeing is the dreaded screen compatibility mode. The dev needs to add the following line to the Android Manifest after the end of the application section:

android:largeScreens="true" android:xlargeScreens="true"/>

He might also have to create higher resolution resources, but I think the default will cause android to scale the current resources up to size...it will be ugly but functional.

See http://developer.android.com/guide/topics/manifest/supports-screens-element.html for more information on supporting large screens.
