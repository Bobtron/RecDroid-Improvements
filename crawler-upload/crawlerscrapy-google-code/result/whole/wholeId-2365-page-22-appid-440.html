Comment #1

Posted on Jul 24, 2013 by Quick Hippo

Well, a good solution for this problem would be a plugin for tasker/locale/llama instead. Usually my approach is to try to find a solution for the root cause of the problem rather than introducing weird workarounds.

A plugin is already absolutely technically possible. CSipSimple has a fully documented api. Some users already created plugins for tasker/locale (see https://github.com/luklanis/csipsimple-plugin ). For now this plugin only allows to enable/disable accounts, but maybe you could ask the developer some enhancements.

On my personal side, I don't develop plugins for commercial apps because I believe in the fact apps should be opensourced and community driven. So I didn't took time to develop this plugin. But if some developers wants to spend time on these topics I'll be pleased to help them. Also, if one day, one equivalent to locale/tasker is released under opensource compliant license, I'll be pleased to create a plugin for csipsimple. BTW, to be complete, I'm developping for a long time such an equivalent, but there is so many things to do, and support of csipsimple replying to each request take me so many time that I never got enough time to release it. (I do that only on my free time : nights and holidays).
