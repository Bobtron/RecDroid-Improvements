Comment #1

Posted on Sep 20, 2010 by Quick Hippo

Not sure that I understand your request.

CSipSimple is already based on the work done in pjsip-jni (it's mentioned on the Project home page). But things have been regenerated using Swig and I modified a lot of code so that I must publish my fork here : There is additional features I added, and also things specific to the android JVM (dalvik). It is also now compatible with pjsip 1.x trunk (while base pjsip-jni is only compatible with 1.0.x trunk).

So would be hard (and a terrible loss of time) to separate my modification in a patch.
