Posted on Aug 22, 2010 by Quick Hippo

You can contact me directly r3gis.3r at gmail dot com (the contact is available on the android market about the author section - don't remember if it's available on the archos market). Btw, as the question is asked, there is a miss, and so updated, the main project page and added a note on the bottom of the page related to contact.

As for porting Froyo on Archos 5 IT, great idea ! I can directly say you that's it's of course ok. You just have to respect licensing terms but it is GPLv3 and should be ok since your project seems to be opensource. (Even if Archos want to integrate the app in their phone/tablets it would be ok too ... but as for now they didn't respond to my question regarding some bugs on the Archos 5IT... I guess they are not interested with a VoIP application yet....)

Just an important point. I consider that for now CSipSimple is still in its alpha phase of the development. Don't know when you plan to land something but not sure that's a good idea to include something still in alpha if your project is already stable. But we can discuss about that in private message if you want.

Besides, let me know if you need some changes in my code to match exactly what you want. I guess almost everything will be fine, but probably we should disable the "Call via GSM" feature for example. I already made some change to match the actual archos 5IT official ROM (for some unknown reason IN_CALL audio stream is not implemented and considered as music stream.... that's a pity because except this point the archos 5IT audio driver implementation is by far the best as far as I can tell).
