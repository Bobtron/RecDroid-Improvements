csipsimple - issue #1451

System pops up auto-close message when CSipSimple was idle for 45 more than minutes

Posted on Dec 22, 2011 by Helpful Elephant

What steps will reproduce the problem? 1. Wait more than 45 minutes 2. sometimes, message "CSipSimple was active for more than 45 minutes and used battery power. Would you like to close CSipSimple?" appears. 3. When choosing "no", CSipSimple continues working.

System message is very annoying. If choosed "yes" by mistake, CSipSimple is being killed by the system and must be restarted manually. Mentioned system message can even pop up, when battery is charging. Who cares about power consumption while charger is connected???

What is the expected output? What do you see instead? System message "CSipSimple was active for more than 45 minutes and used power. Would you like to close CSipSimple?"

What version of the product are you using? On what operating system? Motorola Milestone 2 (Vodafone DE) Gingerbread 2.3.4, CSipSimple 0.03-01 r1108

Please provide any additional information below. Sorry, I don't have any so far.
