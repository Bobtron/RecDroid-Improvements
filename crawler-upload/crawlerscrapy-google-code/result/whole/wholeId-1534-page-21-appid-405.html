Comment #4

Posted on Jul 3, 2009 by Swift Camel

I experienced the same problem after using vcardio, i don't have a test case but provide my workaround if anyone else are affected.

My workaround was to delete the contacts store (Home -> Settings -> Applications -> Manage applications -> Contacts Storage -> Clear data) and resync with Google. Deleting all Gmail contacts from web, then forcing a sync on the Magic, then reimporting everything in Gmail and forcing another sync didn't solve the issue. Likewise contacts created after the vcardio sync also couldn't save icons (or icon was stored under another contact).
