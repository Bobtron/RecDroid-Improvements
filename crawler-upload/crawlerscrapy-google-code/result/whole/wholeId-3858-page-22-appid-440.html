csipsimple - issue #964

Per-account authorized wifi access point list

Posted on May 15, 2011 by Grumpy Bear

In Nokia/Symbian each SIP account must have its own wifi access point. This isn't very convinient (required extra effort for grouping the same account using different wifi APs).

However it allows to use only the specified wifi access points for SIP. E.g. my wifi at home and at my favorite bar are okay for SIP, but my work wifi isn't. Currently it's an 'all in' situation.

You could provide a list of 'authorized' wifi APs per SIP account or global 'wifi okay for sip' list.

Comment #1

Posted on May 16, 2011 by Quick Hippo

This kind of feature should be provided by a plugin. In order to keep the application simple and in order to make this kind of feature much much much more powerful than if it was included into the base app, it's better to have this feature outside in a plugin.

It could be plugins of other apps such as locale or tasker. I'll not do myself one plugin for one of these two apps cause that's closed source app and making a plugin for one of these would mean I encourage my users to buy one of these apps. If there is an opensource project that do same things I'll do plugins for CSipSimple. If anybody else that does not care about users freedom wants to do a plugin for tasker or locale, it will be welcome. CSipSimple has an open api that already allow this kind of plugin.

Read my comments in following issues, you'll se that your feature request, as those of theses issues are more relevant of a plugin : http://code.google.com/p/csipsimple/issues/detail?id=410#c1 http://code.google.com/p/csipsimple/issues/detail?id=586#c1 http://code.google.com/p/csipsimple/issues/detail?id=871#c1

One thing that I have in mind is to start this kind of tasker/locale like but opensource project with a real plugin approach. Unfortunately, I've not really time currently. CSipSimple eat all my free time and I've a real full time job, so my nights and holidays are not long enough to start another big project ;). But if there is an opensource project that do this kind of "context aware" via plugins features, I'll do something for csipsimple.
