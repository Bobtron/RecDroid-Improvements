csipsimple - issue #411

[Enhancement] Option to redirect all GSM calls to voice mail when ongoing VoIP call

Posted on Nov 23, 2010 by Grumpy Elephant

An option to allow the use of USSD codes just before starting a VoIP call to redirect any GSM call to voicemail during the call. This is set back to default.

For example : ussd:**21* sip:corrrespondant number call.... end call... ussd:*#21#

Please check the USSD codes implemented by your carrier. In France we have

Redirect to voicemail settings: Active 61*(voicemail number)(Nb Sec ringing)# [SEND] Unactive ##61# [SEND] (Sonne Toujours) (8) Check *#61# [SEND]

The trick is replacing the voicemail number by the VoIP number

As GSM is almost standard, It may work (as far as I know, there is some Ericsson equipment in Argentina.
