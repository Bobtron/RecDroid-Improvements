android-daisy-epub-reader - issue #34

Variable speed playback for audio books.

Posted on May 29, 2011 by Helpful Panda
  1. Since I read lots of dtb's I'd be interested in a variable speed function in the player since not all full-audio books are recorded/ released in normal speed (for a number of reasons).

Raised on behalf of /laforge who requested this feature on the project's Google Group.

I've set the priority to low at the moment since I doubt the Android MediaPlayer supports this capability. However I'll investigate the options and raise the priority if I think this proposal is feasible.

Comment #1

Posted on Sep 2, 2011 by Helpful Panda

I have been investigating the capabilities of the Android Media Player. I don't think it supports the ability to vary the speed of playback. So currently this request does not seem feasible. If you have ideas and suggestions on how to vary the speed on Android please add them as comments here.
