See here how to convert DP to pixels: http://stackoverflow.com/questions/5959870/programatically-set-height-on-layoutparams-as-density-independent-pixels

So that line should be changed to something like:

int previewSize = (int) TypedValue.applyDimension(TypedValue.COMPLEX_UNIT_DIP, , getResources().getDisplayMetrics()); RelativeLayout.LayoutParams lp = new RelativeLayout.LayoutParams(previewSize, previewSize);
