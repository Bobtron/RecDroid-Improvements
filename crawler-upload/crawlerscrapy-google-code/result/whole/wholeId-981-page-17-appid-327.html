Comment #1

Posted on Oct 16, 2010 by Helpful Panda

I agree the program would be better if it supports the bookshare books, which are in DAISY 3 format. Currently the program only plays DAISY2.02 full audio books.

The current user-interface is very limited, with nothing on the screen when the book is playing. However by using gestures you can navigate between chapters and other sections of the book. The instructions are available on the Menu page (press the menu key on your phone) - essentially, swiping from top to bottom of the screen will go to the next section, swiping from bottom to top goes to the previous section. You can navigate between 'levels' of the book by swiping from left to right to increase the level (which allows you to navigate between sections within a chapter - assuming the book contains levels). If you swipe from right to left you decrease the level, which means each swipe navigates through more of a book (again assuming the book includes several levels).

The current user interface was intentionally basic and written for blind users. At some point, probably after I've added some support for DAISY 3 books, I expect I'll provide a more visual user interface.

Thanks for your comments.

Comment #2

Posted on Dec 29, 2010 by Helpful Panda

A quick update: The user-interface has been improved when playing Daisy 2.02 full-audio books (the only format supported by the DaisyPlayer currently). We still don't support Daisy 3 books of any type - full audio Daisy 3 is likely to be added before full-text books.
