Comment #7

Posted on Dec 23, 2010 by Quick Hippo

The good way is with username@sip.justvoip.com

I tried to create filter for Skype Out using 'Directly Call' if ends with '.com' but in that case it directly opened Skype application

Sounds normal lol :) If you add a filter to "Direct Call" for Skype Out .... it will directly call with Skype Out. You should add a filter to "Direct Call" with for your SIP account (and remove the one on Skype Out).

The thing you have to take care is that Skype integrate a non standard way to the dialer : So if you have skype installed (that's not required to have skype intent working), you will be first prompted to choose between 'dialer' and 'skypeout' In this case you must choose Dialer (and you can activate 'remember this choice'). (If you have mistake once and activated skype you can go in android settings > Application > go to skype application and "remove saved default actions").

Then once dialer is chosen (or automatically if you have "remember my choice" in previous popup), it will be csipsimple turn ... You should have the CSipSimple window with as choice all your sip accounts (justvoip included), dialer and skype (if installed).

Here you can choice justvoip account manually.

In this popup filters applies. So if you add a filter "Direct Call" for your account JustVoip, it should automatically call using your JustVoip account.

Let me know how it goes. I'll retest today (cause that's a fresh implementation and I don't exclude some mistake of mine ;) ). But I quickly tested yesterday and was not so bad.
