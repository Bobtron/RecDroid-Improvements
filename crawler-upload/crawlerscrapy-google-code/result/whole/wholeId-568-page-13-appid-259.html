hmm should work but i haven't testet the wlan thing yet

yeah for 2 emulators you have to modify the line of //IP=10.0.2.2 in sendmessage function in imService just delete the 2 "//" in front of it

and then while running the emulator you have to setup port forwarding let me give me an example for instance user1 logins the emulator-5554 with port number 12345 and user2 logins the emulator-5556 with port number 54321

then issue these commands

adb -s emulator-5554 forward tcp:12345 tcp:12345

adb -s emulator-5556 forward tcp:54321 tcp:54321
