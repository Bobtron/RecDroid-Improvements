geform - issue #69

Grafico exportado do highcharts nao esta codificando corretamente os caracteres

Posted on Feb 13, 2014 by Quick Ox

Ao exportar um grafico do relatorio os caracteres com acentos e/ou especiais estao sendo apresentados incorretamente, mesmo que na pagina web seja mostrado corremante. A image1.jpeg ilustra o problema.

Comment #1

Posted on Feb 13, 2014 by Quick Ox

Acredito ja ter me deparado com esse problema antes e resolvido. Alguma modificacao deve ter desajustado o que era necessario para que o grafico seja exportado corretamente.

Comment #2

Posted on Feb 13, 2014 by Quick Ox

Esse parece ser um problema recorrente, pois no proprio site do Highcharts tem um ponto falando sobre o assunto (http://www.highcharts.com/docs/frequently-asked-questions#non-ascii). O problema deve se resolver garantindo que a pagina onde o grafico esta definido está em utf-8.

Comment #3

Posted on Feb 13, 2014 by Quick Ox

Alterei o arquivo index.jsp que estava em UTF-8 sem BOM para UTF-8. Para fazer essa conversao utilizei o Notepad++ (Encoding > Convert to UTF-8). Parece que isso resolveu o problema (image2.jpg). É importante ficar atento agora com isso e garantir que qualquer modificacao no arquivo nao altere o encoding.
