android-on-freerunner - issue #40

usb storage mode

Posted on Oct 14, 2009 by Happy Bear

On real android phone, once you plug in USB cable, an USB icon appears on the phone, both in status bar and notification area. If you click on the USB icon in notification area, it allows you to switch the USB interface from USB network to USB storage mode. The USB storage mode allows desktop to interact with the SD card directly.

At least MS Windows detects the switch automatically and prompts the "Explore" window.

The same icon is used to change the USB interface functionality back into network mode.

Please implement the same feature for the FR.
