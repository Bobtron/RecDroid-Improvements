Comment #15

Posted on Aug 11, 2010 by Grumpy Kangaroo

I've updated the app to that the search button deals the cards on all the solitaire games. Did this so that I could deal the cards on my Galaxy S when playing Spider. This is still the best version of Solitaire. Anyway, if you'd like a copy of the app let me know: rockymount1@gmail.com I've got an e-mail in to the Ken Magic (owner of this app) to make sure it is OK with him. If you want to do it yourself, you can download the source code, install Java SDK and Android SDK and compile it yourself. You then need to sign it with your own key. Anyway, the line you need to change is in SolitaireView.java at line 520.

Change: case KeyEvent.KEYCODE_DPAD_CENTER: To: case KeyEvent.KEYCODE_SEARCH:
