I guess that's something with the way you build the apk file. Do you sign it with a dev certificate? Do you know that you can directly deploy from the eclipse ADT (android dev toolkit) plugin to a device that is connected in USB. Btw, in emulator you'll get a very very bad call quality due to the fact the emulator driver (at least on linux), is not very efficient. As consequence CPU of qemu will be pledged.
