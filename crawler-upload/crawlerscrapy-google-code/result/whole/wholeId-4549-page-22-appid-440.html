Comment #5

Posted on Oct 21, 2010 by Quick Hippo

Ok For stun as it's not linked to the account setting it is in csipsimple global settings. Go on dialer screen, press menu and -> Settings > Media > STUN. (tick the checkbox and you can fill the stun server with the one of your provider, but counterpath's one should be ok too... stun is not linked to the provider, it's just a service that allow the phone to know his IP in the public network).

However, if you get it worked previously, it's probably not STUN (unless you were trying on wifi previously and then in 3G -> STUN is done to solve NATed network problems, and your 3G network is probably NATed while your Wifi network is probably not NATed).

Another cause could be the fact you installed (or started) another Sip client on your android while using CSipSimple. Sip clients usually listen on the same ports, so they can conflicts if running at the same time (even if running in the background... I recently had the problem with somebody else who had sipdroid running at the same time of csipsimple and was not able to get any sound... cause sipdroid was blocking media ports).

So keep me in touch. If you can produce logs, it can also help me (see HowToCollectLogs wiki page)

Comment #7

Posted on Oct 23, 2010 by Quick Hippo

For dialing rules : Edit the account, press menu, choose filter... and you'll enter a powerful world ;) This interface allow you to : Rewrite numbers dialed from the android native dialer for this account : you can choose "rewrite " begin with 0 by 00. Then when you'll dial from android dialer or clicking a contact ;in the list of choices GSM will not be rewritten and sip will be rewritten. You can also use this interface to ignore some numbers or to automatically use sip (if available ) for others... (all that stuff is for android native, for CSipSimple one as you choose the account in the dialer it's more logical to keep what the user is typing assuming it doesn't need to be rewritten ). You can consult the FAQ wiki page about this feature tell me if you think it's not complete enough and what I should add.
