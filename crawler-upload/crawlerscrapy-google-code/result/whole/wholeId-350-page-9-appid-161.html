android-maps-extensions - issue #35

GoogleMap.onMapLoadedCallback()

Posted on Nov 20, 2013 by Quick Lion

The extensions library doesn't support the new GoogleMap.onMapLoadedCallback().

https://developers.google.com/maps/documentation/android/reference/com/google/android/gms/maps/GoogleMap.OnMapLoadedCallback

This would be a great addition to the extensions library, as it allows the developer to know when the map isn't just tiles on the handset's screen. Will we see this in an update to the extensions library, or is there a workaround to include this easily?

Thank you.
