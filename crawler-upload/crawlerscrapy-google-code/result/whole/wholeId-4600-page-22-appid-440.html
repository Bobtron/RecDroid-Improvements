Comment #7

Posted on Sep 20, 2010 by Quick Hippo

Ok as for sound issue : Here is what you can try to improve things : * Disable all codecs except PCMU (go in settings > media > codecs) * Disable VAD * Disable Echo cancellation * Change the audio frequency (try 16kHz & 8kHz (it's probably the current default value) )

Mic amplifier is a feature that was already existing in 0.00-12. The only thing that change is that in 12-0 there were no interface to change the value ;). What change in current version is the fact there is more codecs and also vad and echo cancellation and frequency default values are not the same. There is also something that fix an issue with samsung galaxy S... it can affect a little bit audio quality but not something significant. So I think that it's due to codec in use. (Some codecs consume a lot of CPU and can produce this kind of issue).

As for initial issue, I close this issue. But you can continue to reply here for audio issue, I'm still notified about changes.
