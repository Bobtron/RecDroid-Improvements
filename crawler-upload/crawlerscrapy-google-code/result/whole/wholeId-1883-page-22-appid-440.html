Comment #1

Posted on Aug 23, 2014 by Quick Hippo

Please ask this kind of question on csipsimple-dev group. Better to share problems with others.

The first problem you might have is the download of silk that is not distributed anymore by skype/microsoft. I've just pushed a fix for that.

It also appears nproc is not available on your platform. I don't know how to install that on mac, but it should be possible as there is other users on the mailing list using mac to build.

Also, it seems your makefile ignores failures and keep continuing (which is weird). I do no see any output for the NDK build step which makes me think it's actually not successful and that's the key step. Without this step no shared library will be produced and thus the app build will not be able to load the library.
