Comment #21

Posted on Mar 14, 2011 by Quick Hippo

That's not a webservice.

That's a webinterface to view datas.

A webservice is something used by an application to ask a webserver for an information.

Basically it's an HTTP POST/GET (I'd prefer it to be over https ) that do an unique query and get a response that can be parsed by an application. For example XML or JSON... but not an entire HTML page.
