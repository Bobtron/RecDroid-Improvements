csipsimple - issue #1531

Homescreen Widget: Status inaccurate -- no "orange" line, always "green" line

Posted on Jan 23, 2012 by Massive Elephant

What steps will reproduce the problem?

Activate account via click on Homescreen Widget OR activate account in application

What is the expected output? What do you see instead?

-- if the status is "green" in the homescreen widget, it should reliably indicate that the account is active and registered (!)

-- however, often the status is green even if the account is not (yet) registered or if it de-registered for some reason (as indicated by a red error message in CSipSimple/Accounts)

-- there should be an orange line instead of the green one to match the status of the account as shown in the application

What version of the product are you using? On what operating system?

003-01 r1108; Android 2.3.3

Please provide any additional information below.

The status line is unfortunately not reliable. The icon in the status bar does not show if there any problems with registering one account either. So, if the CSipSimple icon is shown and the status lines are green, you suppose that all activated accounts are also registered, which is unfortunately sometimes not the case -- but is NOT obvious on the homescreen. (So you need to manually open the application regularly to see if there are any problem.)

The homescreen widgets cannot be used reliably to activate accounts as long as they turn green as soon as you clicked on them. The line should only be green when an account has sucessfully registered.
