Comment #4

Posted on Aug 23, 2012 by Quick Hippo

I've just seen logs :)

Indeed not a problem with the timeout value, so no need to change the wizard.

It seems a problem with codec negotiation. Can you try to go in settings > media > codecs and disable all codecs and leave only PCMU enabled. It will avoid the second round on codec negotiation that goes wrong.
