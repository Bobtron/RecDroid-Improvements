Posted on Sep 22, 2012 by Helpful Panda

Every time I try to place a call using the csip simple betamax plugin, in the exact moment I click on the call button the app crashes. So far it did not work once! I have a Zte Blade, stock rom with Froyo 2.2 Any advice? Thanks

Comment #1

Posted on Sep 22, 2012 by Quick Hippo

As replied in private mail :

Betamax company has stopped their feature to have web callback on server side. It breaks everything in the way to use their API and didn't publish anything to have apps using their webservice about 9 month ago (depends on the clone).

They have broken the web callback and the interface to get the account balance... So unfortunately no way to use webcallback method until they re-enable the old interface or publish a public api that can be used by other apps. (But if you find an application different than the one of Delmont/Betamax company... let me know, maybe someone was able to reverse engineer their webservices).

Apparently their strategy is now to try to force their users to use their application only... In my opinion it's a bad strategy to try to force users to use only a proprietary application for a service... but that's the way it seems they are choosing.
