Comment #2

Posted on Oct 14, 2009 by Happy Bear

Ah, the fonts needs to be installed separately even on the normal android phone...

Here are the instructions for hebrew (haven't tested it on FR as adb-over-usb is not working for me): (taken from http://www.pocket.co.il/community/archive/index.php?t-222538.html)

1) Download and unpack http://firstpost.org/wp-content/uploads/2009/02/hebdroid.zip onto SD 2) adb shell 3) Use following commands mount -o rw,remount -t yaffs2 /dev/block/mtdblock3 /system cat /sdcard/DroidSans-Bold.ttf > /system/fonts/DroidSans-Bold.ttf cat /sdcard/DroidSansMono.ttf > /system/fonts/DroidSansMono.ttf cat /sdcard/DroidSans.ttf > /system/fonts/DroidSans.ttf cat /sdcard/DroidSerif-Bold.ttf > /system/fonts/DroidSerif-Bold.ttf cat /sdcard/DroidSerif-Regular.ttf > /system/fonts/DroidSerif-Regular.ttf reboot

Once I test the procedure on the FR, I will update the ticket, so it can be closed.
