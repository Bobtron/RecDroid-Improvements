csipsimple - issue #594

[Plugin idea] Widget to select the "default" outgoing route

Posted on Jan 15, 2011 by Happy Bear

Please create a nice widget that would reproduce the useful button on the top right corner ot the "Dial" tab in csipsimple interface - the one that shows the current outgoing call route (sip-1, ... sip-n, mobile). This would allow to easily see from your homescreen through which "current default" connection you are going to dial.

Problem 1: Of course, this would be in conflict with "Filters" and with the "Select outgoing route" screen before dialing.

A simple solution would be to have the widget with the options of "sip-1,... sip-n, mobile, use filters, always ask". People that (1) constantly use filters (and don't care about selecting outgoing route) or (2) prefer "always ask" - would not need this widget at all (and they would not put this widget on their home screens).

Another solution would be to add some option in "settings", something like: Select outgoing route: [x] use widget --> the widget would show sip-1,... sip-n, mobile [ ] use filters --> the widget would show only "use filters" [ ] always ask --> the widget would show only "always ask"

... or - the same option in "settings", but the widget would allow all possibilities (sip-1,... sip-n, mobile, use filters, always ask) and adjust the settings according to the selected thing on the widget.

Problem 2: The (presently available) button on the csipsimple interface in the right corner of the dial tab shows only the options of already connected sips. That would of course not be good with the widget, especially for people allowing csipsimple activate "only for outgoing" calls. A better thing, I guess, would be to show all available options, but with "grey'ed"/red options for the (still) unavailable sip connections. And, of course, the widget could show the "current default" outgoing connection with the according color (red/grey'ed for unavailable, yellow for registering, green for registered).

Problem 3: To overdrive things, it would be nice to have a widget option to allow people to choose between "tap to switch route/option" or "tap to show and select available routes/options".

I don't know if other agree, but I guess this would be a real hammer :)

Anyway, thanks for the efforts, A daily user of csipsimple, Horia
