csipsimple - issue #355

Typos in French

Posted on Nov 8, 2010 by Swift Cat

Some typos I found in French:

  1. The simplified configuration screen ("Configuration simplifiée" in French) that you see when running the app for the first time has a typo: In "Profil de disponibilité", option "Toujours disponible" says "Utilisera de la batterie mais vous recevrai vos appels".

"recevrai" -> "recevrez"

  1. In the network options, 3G option: "Votre opérateur DOIT vous lautoriser pour activer cette option"

"lautoriser" -> "l'autoriser"

Same for GPRS and EDGE options (I suppose it's the same string)

  1. In the "media" options: "Server STUN (hôte:port)" and "Server TURN (HOST:PORT)". The second should also use "(hôte:port)"

  2. The last choice in the Media options is not translated "DTMF SIP INFO", "Use SIP INFO to send DTMF".

  3. In "Apparence et comportement" > "Modifications avancées", it says "Rester allumer lorsqu'en ...": "allumer" -> "allumé"

What version of the product are you using? On what operating system? 0.00-15 on T-Mobile G1 running Cyanogenmod 6.0 (Android 2.2)

Please provide any additional information below. None
