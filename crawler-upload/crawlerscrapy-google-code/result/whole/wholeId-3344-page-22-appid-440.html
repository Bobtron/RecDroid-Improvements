csipsimple - issue #1444

No outgoing calls, incomming no audio, Android 4.01

Posted on Dec 19, 2011 by Swift Monkey

What version of the product are you using? On what operating system?

Based on the GPL CSipSimple version : 0.03-01 r1108 on Android 4.01, Samsung Galaxy Nexus

Please provide any additional information below:

It is not possible to make outgoing calls. Csipsimple refuses to dail. Receiving calls is no problem, except there is no audio. Same settings on a Nexus One does work. Target SIP server is a linuxbox running Asterisk. No errors there when trying to dail. Problem occurs on WIFI and 3G connections.
