New interface FCs / goes funky on Galaxy Note HD screen, minor things on others too

Posted on Jan 11, 2012 by Grumpy Cat

The latest interface commits make csipsimple go FC or totally funky on big screen e.g. Galaxy Note Devices.

1) If you manage to get into settings it kicks you out again to a screwed up main screen within 5secs. If you manage to get into netwrok / security settings there is only one option SRTP, no TSL whatever. Most of the time it will just close after a few seconds.
