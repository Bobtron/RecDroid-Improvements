Comment #7

Posted on Mar 27, 2012 by Quick Hippo

Oh, ok, so display name of the sip uri is not present in your contact. Sorry I thought you talked about the display contact name of the remote AoR. : if your contact has something like "User display name" , normally "User display name" is included. But you are actually talking of android contact name of contact associated to this phone number as detected by CSipSimple.

So, indeed, I don't add the resolved contact name from android contacts because I don't know it at the time I create the file : For obvious performance reason, the resolution of contact name is made asynchronously and is retrieved from android contact database after the call media is established. So anything I don't have in the "SIP INVITE" (that is get from the sip server) is not eligible to be added in file name.
