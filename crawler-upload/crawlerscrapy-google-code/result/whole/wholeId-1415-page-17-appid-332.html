Comment #9

Posted on Dec 28, 2009 by Grumpy Dog

Hi,

the build is wrong because the build.prop file must be manually removed from the out directory. I'm not sure if you can do manually but the file to change is /system/build.prop and add the two battery entry

backlight.battery_full=/sys/class/leds/gta02-power:blue backlight.battery_in_charge=/sys/class/leds/gta02-power:orange

Comment #13

Here is how I managed to get the charging LED under cupcake-daily-20091229 : ./adb shell mount -o remount,rw /dev/block/mtdblock6 /system ./adb pull /system/build.prop build.prop

[edit build.prop] backlight.battery_full=/sys/class/leds/gta02-power:blue backlight.battery_in_charge=/sys/class/leds/gta02-power:orange [write build.prop]

./adb push build.prop /system/build.prop

Thanks Michael !

P.S. : using the AUX LED to indicate notifications is OK as far as I'm concerned.
