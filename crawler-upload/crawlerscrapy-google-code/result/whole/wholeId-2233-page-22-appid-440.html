Comment #9

Posted on Nov 14, 2013 by Swift Dog

While digging around to troubleshoot wifi tethering issues with the Nexus 5, I found a solution that fixes the IPv6 issues altogether for all apps on the device that depends on IPv4. Nevertheless, your foresight of having IPv6 support in your app is wonderful. Vonage couldn't resolve it. The instructions below fixed all (I only had 2 issues - wifi tether and vonage mobile app) IPv4 related issues for me.

Also, I'm not sure if this is specific to T-Mobile USA although I'm leaning more to Nexus 5's IPv6 as the default IP Protocol for LTE enabled carriers, but if anyone has a Nexus 5 with other carriers, please chime in with your settings.

These instructions are for the Android OS, not within CSipsimple. 1. Go to settings 2. Wireless & Network > More 3. Mobile networks 4. Access Point Names 5. Select T-Mobile GPRS (fast.t-mobile.com) or your selected carrier. 6. Click on APN Protocol (by default it's set to use IPv6) 7. Select IPv4/Ipv6 8. Exit out of settings.

I will play with CSipsimple's IPv6 setting with the above change and see if I need to keep it enabled or not.
