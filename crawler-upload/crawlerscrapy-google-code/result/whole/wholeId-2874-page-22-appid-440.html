I found this by chance

SIP port : 5060 Registrar : sip.dialnow.com Proxy server : sip.dialnow.com Outbound proxy server : leave empty Account name : your DialNow username Password : your DialNow password Display name/number : your DialNow username or voipnumber Stunserver (option) : stun.dialnow.com

source: http://www.dialnow.com/en/sip
