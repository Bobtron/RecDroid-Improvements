Ok, I reproduced the problem and fixed it. Was indeed something with camera rotation (most devices has the camera sensor rotated of 270°). So was reproduced on each devices like that. I've upgraded the plugin on the website too.

You might also be interested in the new bandwidth parameter I added. Allows to test good quality if network bandwidth is good enough. I did test with profile 3.0 baseline with 1024kbps and result renders pretty good with nexus 7 receiving this good quality stream ;).
