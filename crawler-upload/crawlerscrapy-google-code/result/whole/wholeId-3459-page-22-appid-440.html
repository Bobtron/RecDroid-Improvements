csipsimple - issue #1362

Problem on WiFi connection

Posted on Nov 4, 2011 by Happy Giraffe

Hi! From 2-3 weeks ago I have a problem with the wifi connection. The appliation works well with mobile network connection. The problem exists only with wifi connection. CSipSimple loses the connection to my sip provider server after about 15-20 minutes that my telephone screen is off. I have to disconnect CSipSimple and start it again to restore the connection. It reconnetcts also when I switch off/on the wifi but then I have same problem! I have a htc sensation (sense) with android 2.3.4.

Thanks for support! /Nunzio

Comment #3

Posted on Nov 5, 2011 by Quick Hippo

ok, your sip provider maybe not correctly take register timeout. What you can try is to transform your account into an expert account : long press on the account row, select "choose wizard" and select expert wizard. Then you'll have a setting "register timeout" option. By default it is set to 15min. Maybe you should try a lower value. If it doesn't help, maybe some logs will be usefull. See HowToCollectLogs wiki page. In case you collect logs, start logging, then wait for 15-20min and then stop logging and send the mail with logs, so that I can see how the re-register goes. Thx

If the problem is about wifi access point, there is maybe something on your access point that do weird things. Is that an authenticated access point? Or some public hotspot? If the problem is about the rom, could be something about the way it manages network access. Do you get some big delay when accessing a webpage on first wifi connection? And does the ROM tends to connect you to other open wifi spot?

To be sure, it could be really intersting if you could test on another wifi access point. Regarding logs, I'm almost sure that it does not depends on the app itself. Maybe some settings in the app like locking wifi could help, but that's not a good and safe solution cause it may drain battery faster. And anyway, if my guess is correct, there is something on the wifi access point that may annoy many other apps than csipsimple.
