Comment #1

Posted on Oct 11, 2011 by Helpful Panda

Reiner, Firstly sorry for not replying for 4 weeks, my day job has consumed all my available time despite my best intentions to work on this project :(

I agree your suggestions would be useful for users who can see well. The current user interface is written primarily to work well for blind users, or users with severe visual impairments. My original co-developer is blind and encouraged me to start this project :)

I envisaged the application would include a variety of user-interfaces that would suit the specific needs of various users e.g. for users who are dyslexic. However we currently only have one user-interface. As I'm currently the only developer (and as you can tell only able to work on the project from time to time).

The menu I plan to add soon, and is covered in the following issue http://code.google.com/p/android-daisy-epub-reader/issues/detail?id=49

The user-interface I'd like to add later.

I'll accept and park your suggestion until either I, or ideally another developer, has the time to create the UI you desire. I've set the priority to low, not because I don't value it, but because I want to get various other capabilities completed first.
