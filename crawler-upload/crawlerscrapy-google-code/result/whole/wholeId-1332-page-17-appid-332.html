Posted on Nov 4, 2009 by Helpful Kangaroo

This wallpaper issue along with all the other GUI scaling issues have to do with the resolution.

Android 1.5 has hard coded for the lower resolutions so when it is displayed at 480x640 the background isn't scaled.

This issue along with all the others would be resolved if we lowered the resolution to one supported by Android 1.5 (Or update android to 1.6)

I see fixing these issues as redundant if we are going to change the resolution anyways.
