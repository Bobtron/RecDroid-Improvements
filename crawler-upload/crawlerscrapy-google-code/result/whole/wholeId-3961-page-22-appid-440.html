Comment #1

Posted on Apr 9, 2011 by Quick Hippo

You have probably not implemented the callback in your app that may allow or not a frequency. implement on your java code : on_setup_audio and on_teardown_audio

This method in your java code is responsible for audio routing, audio focusing setting ringer mode also, and so on. (keep in mind that if your code is not released under GPL you can't use CSipSimple source code for that but should do it by yourself).

If the java part claim to support this frequency (for example if bluetooth routing should only claim to support 8kHz), then return pj_success.
