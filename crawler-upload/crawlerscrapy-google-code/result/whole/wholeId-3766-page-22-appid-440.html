Mobyler is a fork of csipsimple.

(see : http://code.google.com/p/csipsimple/wiki/BrandedVersionsOfCSipSimple) They just reuse csipsimple's source code, so obviously it conflicts.

You can (should) use csipsimple instead. CSipSimple will always be more up to date than mobyler and besides it's the root source code used by Mobyler... so no reason to use a degraded version of csipsimple instead of csipsimple (unless you are a mainstream user that doesn't know about csipsimple ;) ).

Besides Mobyler features are included necessarily in csipsimple feature (cause mobyler must redistribute the source code of the modification due to the fact csipsimple is gpl). And csipsimple allow multiple accounts so you can configure a mobyler account into csipsimple without any problem. If no let me know I'll see in source code of mobyler what are the correct setting and add a wizard for Mobyler.
