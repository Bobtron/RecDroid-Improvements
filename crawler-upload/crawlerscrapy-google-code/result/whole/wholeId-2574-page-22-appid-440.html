csipsimple - issue #2258

[wizard] Phone Power

Posted on Feb 24, 2013 by Grumpy Rabbit

Phone Power (www.phonepower.com) is a VoIP company which apparently purchased BroadVoice. (Note that csipsimple already provides a BroadVoice wizard.) Previous BroadVoice customers who encounter persisting problems are being asked to convert their accounts to the Phone Power servers, with new SIP proxies, etc.

Phone Power recommends a minimum registration interval of one hour, with the SIP registration information:

Auth ID: xxxxxxxxxx (telephone number) Use Auth: Yes SIP User ID: xxxxxxxxxx (telephone number) SIP Password: XXXXXXXXXXXXXXX (all uppercase hexadecimal) Proxy: 208.64.8.6:12060

I can sign on with no problem with both a SPA3000 and a SPA3102. Using csipsimple, I immediately get the following error: "Error while registering - Forbidden"

I have reproduced this problem on an LG G2X, a Nexus 7, and a Nexus 4.

Comment #1

Posted on Feb 24, 2013 by Quick Hippo

You can setup in csipsimple the registration interval in the expert wizard : http://code.google.com/p/csipsimple/wiki/ExpertSettingMode?wl=en#Account_settings

It will allow to validate the settings and to be sure it's a problem with the registration interval (it's expressed in seconds in the settings). You can also try to disable "allow contact rewrite" option that might be not supported correctly by some sip providers.

Let me know, if it helps I'll create a wizard for this provider so that it benefit other users of this provider.
