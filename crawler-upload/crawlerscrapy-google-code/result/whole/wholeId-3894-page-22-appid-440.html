Comment #3

Posted on May 1, 2011 by Happy Monkey

BroadSoft uses the following items for endpoint registration:

Authorization User Name Authorization Password User ID (Line@port) Domain / Proxy Outbound Proxy

A wizard would be most helpful.

The fields in csipsimple match up as follows: Account id = User ID (Line@port) Registration URI = Domain / Proxy Realm = Domain / Proxy User name (login) = Authorization User Name Data (password) = Authorization Password Data Type should be plain password Scheme should be digest Proxy URI = Outbound Proxy

The other fields work with the defaults.
