csipsimple - issue #2737

Loses incoming calls with OVH

Posted on Jul 11, 2014 by Quick Bird

I use OVH (which provides SIP) in France.

After a few minutes (the delay seems to be random), incoming calls do not work anymore, and callers get a "can not contact number" answer from the provider ("le numéro que vous appelez est actuellement indisponible", for those who like french :) ).

I tried to work only in Wifi, then only in 4G, changed a lot of settings, broke the app, reset settings, broke the app again, etc. :)

Finally I made a test using ZoiPer, which unfortunately works perfectly :( (this is unfortunate, because ZoiPer has no integration within the system).

Is this a known problem? I guess this can be solver by a parameter somewhere, but which one...?
