csipsimple - issue #2395

TLS Doesn't Work

Posted on Jun 7, 2013 by Quick Bird

What steps will reproduce the problem? 1. Generate a CA certificate using OpenSSL. Self-sign it, copy it to the device, select as CA file within the CSipSimple settings menu, enable TLS and force TLSv1 using expert mode settings. "Check server" and "Check client" are both disabled and encrypted media is disabled (for now)

2.Configure Asterisk server's sip.conf as follows: [general] tlsenable=yes tlscertfile=/opt/etc/openssl/cacert.pem tlsprivatekey=/opt/etc/openssl/private/cakey.pem tlscafile=/opt/etc/openssl/cacert.pem tlsclientmethod=tlsv1

[lizzy] transport=tls ...

  1. CSipSimple successfully registers to asterisk and user lizzy can make outbound calls using CSipSimple but cannot receive calls.

  2. Use OpenSSL to generate a private key and certificate for lizzy and sign it with CA certificate generated in Step (1). Put the files onto the device and enter their respective filenames into the "TLS private key file" and "TLS certificate file" settings.

  3. Exit the settings menu

What is the expected output? What do you see instead?

Expected output: client should stay registered and both inbound and outbound calls should work.

Actual output: CSipSimple deregisters and outbound call no longer work. Asterisk CLI shows SSL error 0 and CSipSimple log shows, "ssl0x630519b0 Error adding private key from '/sdcard/documents/lizzykey.pem'", and, "tlsc0x40c86024 TLS connect() error: Unknown error 483316", immediately after information about a REGISTER request.

Poor workaround: Inbound calls, outbound calls and registration work if the CA certificate and the CA's private key are used instead of the ones generated for the user but this puts the whole VoIP network at risk of intrusion if the mobile device is lost or stolen and prevents giving users time limited access using different certificate expiration dates.

What version of the product are you using? On what device / operating system?

CSipSimple 1.00.00 r2225 Android 4.2.2 Google Nexus 7 3G
