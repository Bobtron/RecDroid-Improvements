csipsimple - issue #1766

SMS through csipsimple app

Posted on May 29, 2012 by Happy Dog

One of my Service Provider is offering SMS service and I tried to send four SMSs from csipsimple app using their service. All four messages were delivered to the recipients but I found that two of them received message with sender id as 'tvi' and two of them received message with sender id as phone number of UK.

Is it possible to create our own sender id (as our own number or alphanumeric code) for sending SMSs so recipient can identify the sender. (My service provider has already provided this option in their application which I use from computer) In case of sender id as UK number, I think its routing problem.

Also I would suggest you to create facility to add signature in SMS so we can add our identity in signature.
