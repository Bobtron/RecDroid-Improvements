Comment #2

Posted on Sep 5, 2010 by Quick Hippo

Interesting.

The "select account" tab correspond to the account setting list? I think that the little popup window is nicest that a tab. Besides, I think that it's not necessary to add a tab to configure accounts since accounts list are just a configuration panel and will be accessed by users only 2 or 3%of time. So accessing by menu is a good way. But the idea of a widget to turn off/on accounts is really nice ! I'll keep it in mind :)

As for in call screen, I think that I'll keep this layout since it match exactly the Froyo "in call" layout (the one that is displayed to users that have GSM on their device). (That's the same for incoming calls). You are probably not used to use it but to answer you slide the green "button" to the right and to decline/discard you slide the red "button" to the left. That's the way it's done in android phones. So it's natural to users I think (however, there is already an option in Settings > UI to use buttons accept/decline if you prefer).
