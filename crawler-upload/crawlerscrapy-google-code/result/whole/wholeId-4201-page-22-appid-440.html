csipsimple - issue #633

Simple 1/0 Switch, without fancy selection field....

Posted on Jan 26, 2011 by Helpful Kangaroo

Hi,

i am able/allowed to use Voip over 3G.

However i dont want to be connected to my Voip all day, plus i dont want to have the extra selection field asking me to choose between mobile call or Voip call whenever using my phone, that is a bit anoying, slowing the phone down and sometimes generating error messages.

Isnt it possible that when i want to use Voip, i simply open the Voip dialer and make my calls and after i am done, i close the application and finished???

Maybe add a radio button in the availability profile "Only when software opened individually"

Comment #1

Posted on Jan 27, 2011 by Quick Hippo

That's already possible. You can simply disable the integration with android either by :

  • 1st solution that you should have already seen at first launch ;) Go in CSipSimple settinsg > Easy settings > Disable "Android integration"

  • 2nd solution that allow you to choose in details what you want to activate in integration : Go in CSipSimple settings > User interface > disable "Dialer integration"

That's it.... Have a look to Settings ... CSipSimple is very tweakable! (If you'd like to tweak it even more read ExpertSettingMode wiki page).

You should also have a look to UsingFilters feature... maybe you are annoyed by this feature cause you don't know that it can be configured using filters... Filters allow you to ignore or activate integration for some patterns... So that you'll never bother anymore with thinking about whether you have to use voip or gsm for a call... Also for slowing down of the integration you should try nightly builds (http://nightlies.csipsimple.com/trunk/); that's maybe better.
