Could you try latest dev version http://nightlies.csipsimple.com/trunk/

And then in expert audio setting : Use webrtc implementation : OFF Use routing api : OFF Use mode api : ON SGS hack : OFF Tone hack : OFF Focus audio : OFF Mic source : default Audio mode for sip call : (does not matter since using mode api :) ).

If some hack helps usually I activate it by default for next release for relevant devices.
