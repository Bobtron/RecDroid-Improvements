Comment #1

Posted on Sep 19, 2013 by Quick Hippo

Hi David,

This kind of error is something raised by the sip server (I mean it's not something invented by the application). The error code you get usually means that the server does recognize the credentials. Depending on the server ( / sip provider) it could be the username, the password or both that are not recognized.

So I would suggest you to extra check the login password entered. Something I often mistake with is the fact some keyboard auto-complete/correct the username or the password or add extra spaces in it. Usually sip servers are strict to letter case. Another common problem/confusion is that the username or password of account created to access the web management interface of the provider could different of the sip username and sip password. It's not the case with all sip provider and I don't if it's the case with your provider, but could be good to check with them what they expect exactly. If they have dedicated instructions (I don't see the link, but sounds there is), maybe good to check with them if their instructions are correct and ask them more details.

On CSipSimple app side, it's unfortunately all I can give as help. From description does not sound a bug in the app but more something with sip provider instructions/configuration.

Best regards Régis
