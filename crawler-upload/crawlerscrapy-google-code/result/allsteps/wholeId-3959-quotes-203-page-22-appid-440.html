Silent Mode when dialing should have an option

Posted on Apr 7, 2011 by Helpful Ox

What steps will reproduce the problem? 1.Make out going call 2.During call csipsimple FC 3.Phone ringing back stuck at silent mode

What is the expected output? What do you see instead? Phone should go back to what mode it was before call but after FC, phone is at silent mode

What version of the product are you using? On what operating system? SGS i9000 2.2.1

Please provide any additional information below. I think there should be an option to allow to choose if they want silent mode for call waiting or not. And optiopn to enable callwaiting. This case if FC phone will ringer mode will not be affected.
