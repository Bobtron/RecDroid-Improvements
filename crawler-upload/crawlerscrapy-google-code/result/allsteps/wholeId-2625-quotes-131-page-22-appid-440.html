Comment #1

Posted on May 6, 2013 by Quick Hippo

It seems to be something with configuration indeed. Did you use the basic wizard? And are you sure there is no extra space in the "server" field (it often happens that the keyboard of some devices automatically add some extra space at the end of words).

It could also be something with the size of the SIP packets vs the network routers. On UDP some routers has limits on the size of packets and might decide to not transmit if too big. It would be surprising on the register but could be interesting to try with compact mode.
