Thanks for the report I didn't know about this new betamax clone.

I'll add that right now in the list of servers.

On your side, what you can do, is to first configure with one of the other betamax server and save it. It will not register first, but then you can switch to Advanced wizard. To do so, long press on the account row, and select "Choose wizard" and then select Advanced. Now, you can edit the account and change the server name to voipyo's one.
