Comment #1

Posted on Sep 2, 2011 by Quick Hippo

I guess you are looking for the SIP proxy feature. If so, the feature is already available in CSipSimple.

To configure it : 1- transform your account into an expert wizard account (long press on account row, and "choose wizard" and select expert wizard (in generic section). 2- scroll down to the Sip proxy field. By default it should be setup with sip:sip.register.ip, just change it to sip:sip.register.ip:9600 (note that the sip registrar field should not be changed if your provider registrar listen on 5060 - sip registrar field is on the top of the expert wizard).

3- if it works, and if your sip provider is a sip provider open to public, let me know I can create a specific wizard for this provider, so that other users with this provider will not have to use the complicated expert wizard ;)
