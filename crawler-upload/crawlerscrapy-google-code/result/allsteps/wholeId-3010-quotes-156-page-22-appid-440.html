Comment #3

Posted on Jul 21, 2012 by Quick Hippo

Please join and ask on the developer google group as said in the How To Build wiki page.

Also search on this google group. This question has already been asked and it's always because something went wrong during the build phase of the native library.

Check that you have in lib folder of csipsimple some libs/armeabi-v7a/libpjsipjni.so and libs/armeabi/libpjsipjni.so

If your device is x86 CPU check that you have libs/x86/libpjsipjni.so

If you have not these files, something went wrong on the native library building process. So, join the developer group and ask with the details on the result of the make command line. Because this probably don't go to the end of the process, and you probably didn't read the result of the command line logs. The last logs are probably stating an error.
