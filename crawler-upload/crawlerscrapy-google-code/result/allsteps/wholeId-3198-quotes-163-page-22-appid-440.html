cannot login correctly with different caller id and username

Comment #1

Posted on Mar 7, 2012 by Quick Hippo

I think that there is some misunderstood about "caller id" term in csipsimple. It's something different from username and from authentication name.

I think that what you are looking for is to setup username and authentication name. In basic and advanced wizard both are plugged to a "login" field. Caller id is just about the display name in contact field. So it's not what you'd like to change. So you're right, the way you configure things you are actually using the same username.

So, to do what you'd like to do you have to convert your account into an "expert" wizard one. You can start from a basic wizard and after long press the account row, choose wizard and select "expert". It will transform basic into expert with same infos (actually all wizard reuse expert fields, it's just differents way to present things).

Once you are in expert mode. The "Sip id" field with something like actually contains the user name. The authentication name is in authentication section (after realm and before password).

The display name (what I abusively call "caller id" because many sip providers calls that this way) is what's come before the "<" in "sip id". For example : "Display Name"

I hope it helps.
