Can you provide a little bit more information about that? The best would be to follow the HowToCollectLogs wiki page and send logs.

Usually text feature is not working because the SIP provider does NOT support it. And nothing can be done on csipsimple side for that, just cry to your sip provider.

Sip message feature, is an extension of the sip protocol that allow to send message over SIP. Then up to your sip provider to : * transmit to another sip client if it's a sip client. * transmit as a SMS using a gateway if they allow it.

But regarding csipsimple, you well guess that I tested the feature before adding this into the app, and that with providers that support it, this feature works... so if you just say "can't text", you obviously guess that it becomes hard to me to understand what is the problem ;).
