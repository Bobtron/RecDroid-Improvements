android-on-freerunner - issue #19

Get 'Connection problem or invalid MMI code' when sending a USSD request

Posted on Sep 25, 2009 by Happy Rabbit

What steps will reproduce the problem? 1. Dial a provider-specific USSD request code (e.g. *777# for balance inquiry on US AT&T GoPhone) 2. Hit send

What is the expected output? What do you see instead? Expected output is a dialog with an account balance message. Instead you see an error dialog with the message: "Connection problem or invalid MMI code"

This was copied from a comment on [http://code.google.com/p/android-on-freerunner/issues/detail?id=12 Issue 12], which deals with display of unsolicited USSD messages.
