Hi,

When I check the certificate manually on my side I get :

verify error:num=19:self signed certificate in certificate chain verify return:0

It makes me think there is maybe some problem with the certificate that makes think android OS it's self signed (maybe not the case on all platform because CA list is not the same). It seems to be confirmed by the logs you get : "TLS connect() error: SSL certificate verification error (PJSIP_TLS_ECERTVERIF) [code=171173]" which means that the ssl certificate verification failed.

Can you try to manually install the CA (or the server certificate) on your android phone. There is usually this option in android settings > security.
