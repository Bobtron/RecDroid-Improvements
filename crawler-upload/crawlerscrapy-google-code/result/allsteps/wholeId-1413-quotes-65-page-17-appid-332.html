android-on-freerunner - issue #36

Browser crashes when access Gears location service

Posted on Oct 13, 2009 by Happy Rabbit

What steps will reproduce the problem? 1. Boot the Freerunner 2. Connect to the network 3. Launch the browser and go to the m.google.com home page. About half the time as it's accessing Gears to get location information it will crash. If you then restart the browser it will work properly until the first browser launch after the next reboot.

What is the expected output? The Google mobile home page.

What do you see instead? The main screen. The browser just disappears with no message.
