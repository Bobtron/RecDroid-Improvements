There is a known bug in some android versions (was the case for nexus 4 for a while) due to a conntrack rule added to apply to SIP. This bug (rom bug) implies TCP packets are blocked by the device conntrack. (More details here : http://code.google.com/p/android/issues/detail?id=32736)

As far as I can see your device could be impacted by the problem (some rom with api 15 where impacted). So maybe it's the case. Do you have more details on the device you are using? I do not find any reference to ROM it uses.
