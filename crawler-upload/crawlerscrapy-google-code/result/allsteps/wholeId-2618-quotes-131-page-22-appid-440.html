Comment #1

Posted on Jan 23, 2013 by Grumpy Panda

please download CSipSimpleVideoPlugin.

in CSipsimple, use "make" to build it, and use "make VideoLibs" to build video plugin.

import CSipSimpleVideoPlugin and CSipSimple into eclipse. then run them.

in CSipsimple's preferences form, you need to enable video. to do so, "video call button" is add to dial panel.

good luck.
