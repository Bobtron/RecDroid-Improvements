Comment #8

Posted on Aug 26, 2012 by Quick Hippo

The standard way of logging use rotating logs of android so harmless to use. However will not produce very interesting information since other logs in the middle could drop logs from csipsimple. The second log method to log to files instead of android log is more interesting for this purpose. It produces file on the sdcard. The only thing is to remember to deactivate it once finish to not remain one entiere month with it activated. For 1 week it will produce reasonable amount of logs. Then you can remove logs files from the sdcard.

To activate this log method : # Enable expert setting mode (if not already done ;) ) : http://code.google.com/p/csipsimple/wiki/ExpertSettingMode?wl=en#General_settings # In user interface section check that log level is set to 4 and enable the "log to file" option. (both option are on the bottom of the list).
