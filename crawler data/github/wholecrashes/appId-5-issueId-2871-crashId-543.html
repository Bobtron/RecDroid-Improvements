Description

When I apply a hole icon pack everything works okay, but if I try to manually changes an icon, at least with a big icon pack it crashes the hole Lawchair.

I try it on a relatively big icon pack (Cornie, with about 6.000 icons more less).

In the V1 it works flawless

Steps to Reproduce

  1. Apply an icon launcher
  2. Long press a particular app to change the icon
  3. Choose a big icon pack (6.000 icons)
  4. Try to open the icon pack to choose another icon

Environment

I tried on a Samsung Galaxy A9 Pro 2016, Android Nougat 7.0 and Samsung Experience UI 8.1
