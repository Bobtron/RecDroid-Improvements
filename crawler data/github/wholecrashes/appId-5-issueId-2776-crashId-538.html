Description

I recently upgraded my devices to Android 8.1 "Oreo MR1". After that, Lawnchair began to systematically crash on start. I don't know if the bug appears if the device is already running Oreo MR1 when Lawnchair is first installed.

Workaround

If I go to my device's settings and manually grant the Storage permission to Lawnchair, it starts functioning normally again.

Expected Behavior

Lawnchair should have displayed my home screen, as it did before the upgrade.

Actual Behavior

After entering my PIN code/unlock schema just after the upgrade, I saw a pop-up saying that Lawnchair crashed and a menu asking me to choose between the launchers installed on my devices. Same thing after selecting Lawnchair in the menu, whether it popped up after a crash or by pressing the Home button.

Steps to Reproduce

  1. Upgrade to Android 8.1 "Oreo MR1"
  2. Boot the device
  3. Unlock the device

Environment

  • Device: Nexus 6P / Pixel C
  • Android version: 8.1.0
  • Launcher version: 1.1.0.1655
  • Rom: Stock Rom
