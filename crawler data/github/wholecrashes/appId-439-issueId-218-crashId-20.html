Expected behavior

Application is expected to work perfectly well on any screen orientation, and save film without crashing.

Actual Behavior

How to reproduce

  • Download and install the application here
  • Switch on your data connection
  • Launch the application
  • Click on the add button (+)
  • Select DM
  • Input in some figures
  • Click on Save film (Film speichern)
  • While it is loading, rotate your screen

Then app crashes

Application version: v1.4.1.0
Operating system: Android v7.0
