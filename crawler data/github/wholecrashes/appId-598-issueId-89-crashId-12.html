Expected behavior

Application is expected to work perfectly well, even after minimizing it.

Actual Behavior

After selecting a movie, then minimizing the app, it causes app to crash.

How to reproduce

  • Download and install the application here
  • Switch off your data connection
  • Launch the application
  • Select a movie, in any section of the application
  • Minimize the application, then the app crashes.

Application version: v1.3.1
Operating system: Android v7.0
