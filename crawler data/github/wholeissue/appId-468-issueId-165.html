I'm looking to see if I can use your app on all android phones and not just the zuk. I'm trying on my pixel 2xl on android 8.1 but the recent app list appears to be empty. After inspection of your code I think this is due to your use of "ActivityManager.getRecentTasks()" which is deprecated since lollipop and since oreo only gives the current app id (I think). There are however other ways to get the running apps using "usage access".
