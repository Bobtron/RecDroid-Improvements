QuantumBadger Jan 4, 2015

Owner

The tests are for web pages, and the gains come from the loading of resources (images, CSS, etc). I doubt there would be any performance gain when requesting a JSON document with no ancillary resources.

http://www.guypo.com/not-as-spdy-as-you-thought/

Some fairly big changes would need to be made to the cache manager, which currently uses the Apache HTTP library, to handle cookies/POST data/etc over SPDY. Minor changes may need to be made to the cookie storage in the account manager.

I'm also slightly wary about bringing in libraries of questionable quality to implement proprietary, experimental protocols.

So I'd be wary of adding this even if someone submitted a pull request, due to the risk of introducing bugs. Sorry :(
Owner
