Some code to blur an image:

final RenderScript rs = RenderScript.create( myAndroidContext );
final Allocation input = Allocation.createFromBitmap( rs, photo, Allocation.MipmapControl.MIPMAP_NONE, Allocation.USAGE_SCRIPT );
final Allocation output = Allocation.createTyped( rs, input.getType() );
final ScriptIntrinsicBlur script = ScriptIntrinsicBlur.create( rs, Element.U8_4( rs ) );
script.setRadius( myBlurRadius /* e.g. 3.f */ );
script.setInput( input );
script.forEach( output );
output.copyTo( photo );

http://android-developers.blogspot.fr/2013/09/renderscript-in-android-support-library.html

Reopening this issue for V3.
We're going to remove the blur effect and create the collapsing toolbar layout that was presented originally. The toolbar layout will feature a tab layout with all the menus (instead of an ExpandableListView).
The collapsing toolbar layout will be 300dp-tall with an image slideshow with the cover + screenshots (so there won't be a "Screens" tab). When clicking on a screenshot, it opens fullscreen with the PhotoView lib (pan and zooming enabled) inside a SnapPagerHelper RecyclerView.

All these changes are directly linked to #118, #75 and #76.
