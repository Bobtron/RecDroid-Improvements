Shuttle version:

v1.6.7 (Upgraded)

Device, OS:

Device: Xiaomi Redmi Note 3 Pro International Edition Kate
OS, ROM: 7.1.2 LineageOS 14.1-20170726-NIGHTLY-kenzo

Description of bug:

When adding folders to a new playlist "Gathering songs" process never ends. I have to kill Shuttle in order to continue using it.

Steps to reproduce:

  1. Go to Library
  2. Go to "FOLDERS" tab
  3. Navigate to SD card
  4. Navigate to folder containing music files (for example my folder has 11 FLAC files)
  5. Press three dots on a folder to open menu
  6. Press "Add to playlist" > "New"
  7. Enter playlist name
  8. Press "SAVE"

Expected outcome:

Selected folder content is added to a new playlist quickly.

Observations/Actual Result:

A dialog "Gathering songs..." is shown continuously, process never ends. Doesn't matter the number of music files that the folder contains.
