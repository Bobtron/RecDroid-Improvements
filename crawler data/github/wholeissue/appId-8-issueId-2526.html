ArchangeGabriel Apr 24, 2018

Contributor

  1. No, once you’ve put one version in /system/priv_app/, you’re set, you can then upgrade normally (through F-Droid, manually…). The system will detect the system app and the updated version, and will launch the updated one with system privileges. And the APK name does not matter. ;)

  2. Don’t know about Uber, but the maps implementation is not full yet. Displaying a map should work, but you cannot have overlays, pick a place, etc.

n76 Apr 24, 2018

Some apps are still using Maps API v1. microG gmscore has Maps API v2 built in, for v1 you need to flash a separate ZIP file that you can find at https://github.com/microg/android_frameworks_mapsv1/releases

No guarantee it will help, but it does with some apps.
