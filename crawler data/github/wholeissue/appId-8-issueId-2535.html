AreYouLoco May 23, 2018

In F-droid you have packages from the two repositories. One from standard fdroid repository signed with F-droid key and one signed with different key from microg repository. If you have version from fdroid installed then trying to install the ones from microg repo will result in warning in fdroid app. You have to uninstall the package you have installed and then install the latest one from microg repo. Thats version starting with 0.24-111*. Packages from fdroid have version numbers 1.6.* I attach screenshot.
