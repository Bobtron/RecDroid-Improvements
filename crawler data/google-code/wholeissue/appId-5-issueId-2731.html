TheShark27 commented Sep 23, 2017

Description

In the drawer, if an app gets long pressed and dragged, there will be a little graphical glitch: the actual icon that gets dragged will act like normal, but the space where it got taken from won't remain blank, but the icon will reappear there for a fraction of second.

Expected Behavior

That space should become blank, because the user should think that the app has been actually dragged.

Actual Behavior

It seems like the icon dupes doing that, and it's kind of weird. Just a little glitch.

Steps to Reproduce

  1. Open the app drawer
  2. Long press any app and start dragging it.
  3. You will notice the same app briefly flashing in the place it got taken from.

Environment

  • Device: Moto G⁴ Play XT1602
  • Android version: 6.0.1
  • Launcher version: alpha 1.1.0.1243
  • Rom: stock
  • Build number: MPIS24.241-15.3-21
