Description

When downloading a new app from the Play Store, the app isn't being added to the launcher pages. We have to add it manually.

Expected Behavior

Icons should be added automatically to the launcher when they finished downloading. This issue wasn't happening before I updated my device ROM to OxygenOS Android 8.0.0 beta. The issue isn't happening on OnePlus Launcher.

Actual Behavior

The app isn't being automatically added.
