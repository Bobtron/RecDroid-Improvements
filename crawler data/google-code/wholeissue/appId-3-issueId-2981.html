crossan007 Jan 1, 2018

It looks like this would be possible using the merge endpoint:

You can perform a merge using the merge endpoint: https://developer.github.com/v3/repos/merging/8

To match the behavior of the Update branch button, which is to merge master into your branch, you would pass these args:

{
 "base": "your_branch",
 "head": "master",
 "commit_message": "Merge master into branch"
}

From: https://platform.github.community/t/update-branch-button-equivalent-in-the-api/4260
