marverenic May 18, 2018

Owner

One big concern I have about this being overwhelming is that 99% of users don't care about attributes of the files themselves (i.e. last modified and file size), so at best these options would get in the way for most users. The other thing is that users can already browse by albums/artists/genres which is a simple (yet admittedly not fully comprehensive) way of browsing by these attributes.

I just don't see how a sort option could be added in a meaningful way without adding noise to the main interface. The file browser, however, I definitely understand having this option – most people who will use the feature will be advanced users who do care about file attributes and sorting contents by these fields.

Let me know if I'm missing a use case here, but I think this only makes sense for the file browser.

marverenic May 19, 2018

Owner

You can create a smart playlist that does exactly this. Just tap the "+" FAB on the playlists tab, select "Smart playlist", give it a name, and leave the default rule of "Song name contains [blank]". This should create a smart playlist with all of the music in your library. Then open the new playlist, tap the sort icon in the top right, and click "Sort by date added."

This playlist will automatically stay up-to-date and will retain this sorted order.
