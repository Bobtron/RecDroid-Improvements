haunma Dec 4, 2017

My apologies. I did not have the latest version of microG installed after all. It's actually quite difficult to be sure of this when you install from F-Droid, as if you mistakenly install the stable version before enabling expert mode and unstable updates, F-Droid will insist that you do in fact have the latest, and nothing else is compatible. To see the snapshot/unstable versions then, you have to first uninstall. Just mentioning this here in case others are similarly confused. I read all of the comments and was still certain that I had the correct version. Wrong!

With the latest snapshot I was able to add a google account and sign in. Still having problems with the google play store (patched version from shadow53.com) crashing on start due to a permissions issue on libcronet. I'm pursuing that with the zip package maintainer as it's not a microG issue per se.
