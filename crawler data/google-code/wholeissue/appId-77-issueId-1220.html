kaaholst Jul 21, 2017

Hhmm... sounds stange
It works for me (LMS 7.7.2).
You can try the filter function from eg. the Album list, it uses the same internals as the search function.
Select the Album list / Menu / Filter, and fill in the search text. Results here shall be the same as you get from using the search function. If this doesn't work, my guess is that this is a server issue.
