The Google Weather API does not use the Google app but instead uses the Awareness Api which is done via Google Play services. This data is cached for around 3 hours by the API or until the temperature changes by 3 degrees, before new weather data is downloaded.

This API also does not allow manual location entry, it only works by using your GPS to get the location.

The actual Google Weather API is not available for public use unless the package name is changed to the Pixel Launcher.

The devs won't be able to do anything about this until Google changes how the API works or by making the retrieval of weather data from the Google app public.
