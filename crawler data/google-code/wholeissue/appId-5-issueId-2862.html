Note this is a FR not a bug.
The FAQ section of the homepage directed me here to create feature requests.

Description

The famous and much appreciated launchers from HTC used to have a few useful options at the top of the app drawer:
example
FR 1. a menu button to sort the apps 1) Alphabetical 2) by date 3) custom.
FR 2. a settings button to open the Launcher settings.
FR 3. The same seetings button could give direct access to App drawer settings such as grid (columns x rows).
FR 4. a Play Store button to open the Play Store directly.

Expected Behavior

Have a look at this example.
The search bar takes up the full width of the App Drawer. Add 2 buttons to support the features as described above.

Actual Behavior

Currently the top of the App Drawer consists entirely of a search bar.

Steps to Reproduce

Since this is not a bug, there is nothing to reproduce.
